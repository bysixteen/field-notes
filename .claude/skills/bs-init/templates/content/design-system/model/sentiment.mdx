## Definition

Sentiment is the **emotional category** of a component. It is a composition decision — chosen by the developer when placing the component in a layout.

## Values

| Value | Meaning |
|-------|---------|
| `neutral` | Default — no emotional charge. |
| `primary` | Brand attention. |
| `positive` | Success / confirmed / completed. |
| `warning` | Caution / in-progress / needs attention. |
| `danger` | Error / failed / destructive. |

Replace this list with the sentiment vocabulary your project actually uses. The `default` value is the one omitted from variant names per the cap policy in `references/dimensional-mapping.md`.
