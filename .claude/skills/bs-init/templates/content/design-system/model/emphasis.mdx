## Definition

Emphasis is the **visual weight** of a component within its sentiment. The same `primary` button can be a high-emphasis filled call-to-action or a low-emphasis text link.

## Values

| Value | Meaning |
|-------|---------|
| `high` | Strongest visual presence — filled / saturated. |
| `medium` | Default — outlined or moderate fill. |
| `low` | Quietest — text-only or ghost. |

Replace this list with the emphasis levels your project uses. Three levels is conventional; some systems use only two.
