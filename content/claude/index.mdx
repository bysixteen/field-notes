This section documents patterns, research findings, and practical frameworks for working effectively with AI coding agents. The goal is to move beyond ad-hoc prompting toward systematic, repeatable workflows that compound in value over time.

## What You'll Find Here

### Concepts

**[Context Engineering](/claude/context-engineering)** — The discipline that replaced prompt engineering. Strategies for curating the optimal set of tokens during LLM inference, including runtime retrieval, progressive disclosure, compaction, and sub-agent architectures.

**[CLAUDE.md Playbook](/claude/claude-md-playbook)** — The `CLAUDE.md` file is the primary interface between your codebase and an AI agent. Research shows that having one improves agent efficiency — but only if it is human-written, minimal, and actively maintained.

**[Agent Efficiency Research](/claude/agent-efficiency)** — Two prominent studies reached seemingly contradictory conclusions about the value of agent context files. This page reconciles them: context files work, but authorship and size matter enormously.

**[Research to Validation Pathway](/claude/research-to-validation)** — A five-step process — Research, Synthesis, Prototype Creation, Validation, Outcome — that keeps agent-generated work traceable back to the decisions that motivated it.

### Practical

**[Claude Code Setup](/claude/claude-code-setup)** — Practical guide for configuring Claude Code on a new project: CLAUDE.md sizing, rules with glob activation, hooks for automation, and session management.

**[Skill-Based Automation](/claude/skills-patterns)** — Common skill categories observed across projects, structural best practices, and the design principles behind effective skills.

**[Workflow Architectures](/claude/workflow-architectures)** — Survey of proven workflow patterns: Command → Agent → Skill, Holy Trinity, TDD-first, spec-driven, wave execution, and parallel review.

## Guiding Principles

- **Minimal over maximal.** Less context, delivered precisely, outperforms walls of text.
- **Human-curated over generated.** Agent-written documentation about how to use agents creates a feedback loop that degrades quality.
- **Compounding over static.** The best context files grow through real corrections, not speculative additions.
- **Traceable over ad-hoc.** Every prototype should connect back to the research that motivated it.
